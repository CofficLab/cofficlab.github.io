## 什么是 Web Component

Web Component 是一组标准，允许开发者创建可重用的自定义元素，并将其集成到 Web 页面中。

## JuiceEditor 中的 Web Component

JuiceEditor 将复杂的渲染逻辑隐藏到一个 Web Component 中，对于使用者来说，只需要在页面中加入这个标签即可渲染出一个复杂的富文本编辑器：

```html
<juice-editor></juice-editor>
```
